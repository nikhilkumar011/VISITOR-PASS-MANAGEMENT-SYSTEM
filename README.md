# VISITOR PASS MANAGEMENT SYSTEM  <br/>
A full-stack **Visitor Pass Management System** built with **MERN stack** that allows organizations to manage visitor requests, approvals, QR-based entry, and email notifications efficiently.

### Frontend

### Backend
- Node.js

# Project Setup Guide

-Install Backend Dependencies

-npm install

-Install Frontend Dependencies
